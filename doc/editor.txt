Edit area
=========

All modes

	MMB		pan map
	Arrows		scroll map
	Space		toggle lighting

Tile mode

	LMB		place tile
	Shift + LMB	pick up tile
	RMB		remove tile

Lights mode

	LMB		create light
	LMB drag	move light
	RMB		remove light

Object mode

	LMB		create object
	LMB drag	move object
	Shift + LMB	pick up object
	Ctrl + LMB	raise object
	Ctrl + RMB	lower object
	RMB		remove object

Start points mode

	LMB		create start point
	LMB drag	move start point
	RMB		remove start point


Selector bar
============

	Mouse wheel	scroll list
